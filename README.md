# Hajj Kiosk App

A self-service touchscreen kiosk application designed for Hajj pilgrims to book
Tawaf or Sa'ee time slots, make payments, and collect wristbands — without
needing any assistance.

## Features

-  Bilingual support — Arabic (full RTL layout) & English
-  Accessibility-first — auto height-adjusting screen & dedicated
  special needs counter
-  Multiple payment methods — NFC, card, and cash
-  Session management — idle timeout, 5-minute session limit,
  and auto-clear confirmation screen
-  7-step booking flow: Language → Service → Time Slot →
  Summary → Payment → Confirmation → Rating
-  Multimodal guidance — wristband instructional video on confirmation screen
-  Built for 1024×768 touchscreen kiosks — no keyboard, no mouse

##  Tech Stack

- **Framework:** React 19 (pure frontend, no backend)
- **Logic:** Client-side only — single JavaScript file
- **Deployment:** Vercel
- **Languages:** Arabic & English active — 6 more built-in,
  enabled without code changes

##  Academic Context

Developed as a Human-Computer Interaction (HCI) course project.
Design decisions are grounded in PACT analysis, Norman's affordance theory,
Gestalt principles, and psychological principles of perception and memory.
